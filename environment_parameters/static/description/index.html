Use Environment in parameters

Do the parameters more secure with Environment Variables

Just use self.env['ir.config_parameter'].get_param('key') and if this key is in environment, config-file or in System Parameters - you will get it.

If you want to use specific environment prefix for the parameter, use self.env['ir.config_parameter'].get_param('key', env_prefix='staging') and it will try to search for 'staging_key' in os.getenv
